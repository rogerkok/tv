## Comment installer
- git clone https://github.com/wasstelmali/webtvgenioos.git ou télécharger le zip.
- composer install
- cp .env.exemple .env
- php artisan key:g
- importer la base de donnees gnios(1).sql 
- php artisan serve 
- Pour aller a l'admin, on fait http://127.0.0.1:8000/admin .
